長時間待在不通風的房間裡，空氣裡氧氣越來越少、二氧化碳越積越多，身體發現「氧氣不夠用」了，就會啟動一連串的「救急反應」，但這些反應做久了，反而會害到心臟跟腦袋。

簡單的因果過程

  1. 房間不通風 → 氧氣少 + 二氧化碳太多
    → 血液變酸（醫學叫「呼吸性酸中毒」）
    → 紅血球變得硬硬的、不容易彎曲，像乾燥的橡皮擦，擠不過細小的血管。

  2. 身體想補救缺氧
    → 腎臟拼命製造「造血激素」（EPO）
    → 紅血球變超多（像血變濃了）
    → 血液變得又濃又黏，像稀飯放太久變稠了，流不順。

  3. 血液又濃又黏又硬，還容易黏在一起（像一串紅豆串）
    → 血流變慢、阻力變很大
    → 全身血管（特別是腦跟心臟的血管）都很吃力。

對腦袋的傷害（容易中風）

  • 腦子的小血管本來就很細，黏稠的血擠不過去 → 小地方長期缺血、缺氧。
  • 血流慢容易在小血管裡形成小血塊（微血栓）。
  • 血管內壁也會受傷、變得不光滑 → 更容易真的堵住 → 腦中風（半身不遂、講話不清楚、突然跌倒等）。

對心臟的傷害（容易心臟衰竭）

  • 血液太黏，心臟要把血「擠」出去就變得很費力 → 心臟長期像在推很重的門，肌肉變厚（心肌肥厚）。
  • 肺部也缺氧 → 肺的小血管收縮 → 肺動脈壓力變高 → 右心室（幫肺打血的那一半心臟）負擔超重。
  • 心臟一方面要用力推血，一方面氧氣又不夠 → 心肌慢慢缺氧、受傷、變弱 → 心臟衰竭（喘不過氣、水腫、很容易累）。

一句話總結整個壞循環

不通風 → 長期缺氧 + 二氧化碳太多
→ 血液變酸 + 紅血球變多 + 血變濃黏
→ 血流不順 + 血管阻力變大
→ 腦袋容易中風 + 心臟容易衰竭

容易記住的簡單預防提醒

  • 房間要常開窗通風，或用好一點的空氣清淨機。
  • 睡覺不要悶在密閉小房間，尤其是冬天緊閉門窗又開電暖。
  • 如果常覺得早上頭很重、很累、臉色紅紅的（但不是健康紅潤），或是容易喘、走幾步就累，建議跟醫生說，檢查一下血是否太濃、肺跟心臟有沒有問題。
  • 保持規律活動（慢慢走路就好），不要整天坐著或躺著。
